O que é a ideia ?

Trata-se de um projeto voltado as práticas de facilitação para reuniões diárias de times. Imagine você o quão monótono torna-se a realização rotineira das reuniões de 15 min. Já se perguntou um dos motivos que levam equipes a abandonarem essa reunião com o passar do projeto ? Pois é, a ideia aqui é dar uma energizada nas coisas e tornar essa rotina um pouco mais divertida.

Qual a motivação do energizer ?

A prática de energizer é aplicada com o intuito de resgatar a energia da equipe e trazer sua concentração através de dinâmicas e jogos.

Energizer Game Champions ?

O Energizer Game Champions é um projeto Open Source criado para estimular o processo de energização das equipes antes das reuniões e cerimônias do SCRUM. Trata-se de uma aplicação simples para monitoramento de ranking e conquistas do time.

O que você encontra nesse projeto ?

  • Práticas de TDD (Test Driven Development);
  • Práticas de BDD (Behavior Driven Development);
  • Bootstrap (template: SB Admin);
  • Maven configurado para build no Jenkins, Sonarqube e Artifactoy;
  • Vraptor (criação de serviços para consumo em app's)
